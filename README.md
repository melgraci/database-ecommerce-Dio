# Database E-commerce DIO

Projeto de modelagem e implementação de banco de dados MySQL , desenvolvido como simulação de uma loja de e-commerce.
Descrição

Este projeto tem como objetivo representar a estrutura de um sistema de loja virtual, contendo tabelas relacionadas a:

* Produtos
* Clientes
* Pedidos
* Fornecedores
* Vendedores
* Estoque
* Relações entre produtos e fornecedores

O banco foi desenvolvido utilizando **MySQL** e contém scripts para criação das tabelas, inserção de dados e consultas SQL.

--Estrutura do Projeto

```
database-ecommerce-Dio
│
├── schema.sql
├── inserts.sql
└── queries.sql
```

**schema.sql**
Script responsável pela criação do banco de dados e das tabelas.

**inserts.sql**
Contém os comandos para inserção de dados nas tabelas.

**queries.sql**

Arquivo com consultas SQL utilizadas para recuperação e análise dos dados.
-- Tecnologias utilizadas

* MySQL
* SQL
* Git
* GitHub


-- Objetivo do projeto

Projeto desenvolvido para prática de conceitos de **banco de dados relacional**, incluindo:

* Criação de tabelas
